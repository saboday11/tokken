Справка

1. Парсинг

Для парсинга нужен 1 рабочий аккаунт
Инвайт должен вида "https://discord.gg/", иначе парсер будет работать некоректно

Коды ошибок:
  • 1 - ошибка с прокси, смените/включите прокси
  • 404 - неверный инвайт
  • 400, 129 - аккаунт заблокирован на этом сервере/инвайте
2. Рассылка

Чтобы спамить пользователям сервера, вам нужно спарсить его. Если нужен спам сразу по нескольким серверам, то просто объедините несколько списков пользователей.
Текст сообщения:
"{hello|hi}" - отправится случайным образом либо "hello", либо "hi"
"[rand]" - случайное число от 1000 до 10000
"[username]" - username собеседника
"[discrim]" - дискриминатор собеседника
"[tag]" - тэг собеседника
"[myusername]" - username аккаунта
"[mydiscrim]" - дискриминатор аккаунта
"[mytag]" - тэг аккаунта
"[@tag]" - тэг собеседника в виде ссылке: "@username"
"[@mytag]" - тэг аккаунта в виде ссылке: "@username"
Максимальная длина текста - 2000 символов.
Автоматическая смена аккаунтов:
Вы указываете количество потоков, выбирая аккаунты, указываете аккаунты для смены и устанавливаете нужное количество сообщений.
Пример: Вы ставите 5 потоков, 1 аккаунт для смены и 5 сообщений, значит с одного аккаунта будет отправлено пяти пользователям 1 сообщение. Всего = 5 сообщений * 6 аккаунтов = 30 сообщений. После того как отправится 30 сообщений, рассылка сама завершится.
Если включить опцию поочередной смены аккаунтов, то рассылка завершится только после блокировке всех аккаунтов, либо когда закончится список пользователей
Задержка: при выключенной опции - от 5 до 10 секунд. Советуем ставить от 0 до 3 секунд.
Оповещения о сообщениях: при получении новых сообщений, вы получите уведомление слева, в котором будет кнопка для открытия чата с пользователем.
Если в логах идут только не успешные логи, проверьте заинвайчены ли аккаунты на сервера. Если нет - проверьте доступность инвайта, может быть такое, что админы заблокировали инвайт, либо аккаунты на сервере
Во время спама чёрный список может обновляться не сразу, это произойдет чуть позже.
Спам по серверам:
Чтобы запустить рассылку по каналам, необходимо спарсить доступные каналы - укажите рабочий инвайт сервера и запустите парсинг.
После вы получите все каналы, на которых можно писать сообщения. При редактирование списка каналов можно увидеть на каких включён медленный режим (slowmode) и его значение (от 5 сек. до 6 часов)
Текст сообщения также можно рандомизировать, доступна конструкция "@everyone" (если на текущем аккаунте это не запрещено).

Коды ошибок:
  • 1 - ошибка с прокси, смените/включите прокси
  • 129 - аккаунт заблокирован на сервере
 End_code:
  • 1, 2 - указанное количество сообщение успешно отправлено
  • 4 - список пользователей закончился
  • 7, 11, 5 - валидные аккаунты закончились, спам завершён
3. Аккаунты

При наличии токена можно указать только его ("Добавить" -> "Добавить несколько").
Если при возобновлении сессии ничего не происходит или возвращается ошибка, установите последнию версию Java (и на Mac OS JDK) и Chrome.
Если при смене юзернеймов на аккаунтах возникает ошибка ("you change your name too often..."), возможно Discord установил временный лимит на изменение имени аккаунта, это должно исправиться спустя некоторое время.

Коды ошибок:
  • 429 - слишком много запросов с одного IP, поставьте/смените прокси
  • 400 - инвайт/сервер заблокирован для текущего аккаунта
  • 404 - сервер не найден, либо недоступен для текущего аккаунта
4. Лимиты

Спам по пользователям:
Украинский номер - 7-10 сообщений
Латвийский, Финляндский, Швейцарский, Эстонский, Литвийски, Польский, Кыргызстанский, Румынский номер - 25-30 сообщений
Казахский номер - 4-8 сообщений
Русский номер - 1-3 сообщения
Спам с инвайтом в тексте:
Украинский номер - 2 сообщения
Латвийский, Финляндский, Швейцарский, Эстонский, Литвийски, Польский, Кыргызстанский, Румынский номер - 13-20 сообщений
Казахский номер - 3-4 сообщения
Русский номер - 1-2 сообщения
Спам по серверам: не более 5 сообщений в секунду
Инвайтинг: не более 100 серверов в сутки
5. Прочее

После бана аккаунта все сообщения остаются, он также останется на всех серверах, к которым присоединялся.
Для работы нужно 10-15 индивидуальных прокси, подходящие сервисы: proxyline.net, proxy.house, myprivateproxy.net
В настройках можно указать таймаут при временной блокировке, это происходит когда с одного аккаунта отправляется слишком много новых личных сообщений: Коды ошибок:

  • 401, auth, 987, 451, 455 - нерабочий аккаунт.
  • 1, proxy, 782 - нерабочие прокси, либо сервер недоступен с текущего IP.
